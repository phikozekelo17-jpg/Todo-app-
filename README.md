# 📝 To-Do List Web App

A simple and interactive To-Do List application built using HTML, CSS, and JavaScript. This project allows users to manage daily tasks efficiently with a clean and user-friendly interface.

## 🚀 Features
- Add new tasks
- Mark tasks as complete
- Remove tasks by double-clicking
- Simple and responsive design

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation)

## 📚 What I Learned
- Working with the DOM in JavaScript
- Handling user input and events
- Creating interactive web applications
- Structuring small front-end projects

## 📸 How It Works
- Enter a task in the input field
- Click **Add** to add it to the list
- Click a task to mark it as complete
- Double-click a task to remove it
